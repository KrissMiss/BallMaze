# BallMaze
This is a logic maze game for mobile devices with touch input and a project compiled for android.
To test this project download <a href="https://drive.google.com/file/d/1bOnDDDG03t1zwuQf699_p1V49Aeu9cs0/view?usp=sharing">ballMaze.apk</a> or download zip-archiv–µ and start in Unity using Device simulator or Unity Remout with usb cable on your smartphone :)
